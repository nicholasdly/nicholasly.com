# nicholasly.com

My personal website and blog, which acts as a public journal of sorts for sharing and thinking out loud. Bug reports are welcome.

## Features

- Super fast performance
- Accessible (Keyboard/VoiceOver)
- Responsive (mobile ~ desktops)
- SEO-friendly
- Light & dark mode
- Fuzzy search
- Draft posts & pagination
- Sitemap & RSS feed
